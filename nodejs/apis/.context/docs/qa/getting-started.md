# How do I set up and run this project?

## Getting Started

### Prerequisites

- Node.js (LTS version recommended)
- yarn

### Installation

```bash
# Clone the repository
git clone <repository-url>
cd apis

# Install dependencies
yarn install
```

### Running

```bash
# Development
npm run dev

# Production
npm run build && npm start
```
